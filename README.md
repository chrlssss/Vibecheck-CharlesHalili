# VibeCheck API Project
**Author:** Charles Halili  
**Course:** CPE 411L

## Project Description
A full-stack web application that provides random fortunes, jokes, and vibe checks via a Node.js Express API.

## How to Run
1. **Backend:** - Open a terminal in the `backend` folder.
   - Run `npm install`.
   - Run `node index.js`.
2. **Frontend:**
   - Open `frontend/index.html` in your web browser.

## Features
- Random Fortune and Joke endpoints.
- Smash counter tracked via POST requests.
- Interactive UI with mood-based responses.
